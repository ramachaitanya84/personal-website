# Design Rules

Aesthetic:
- Minimal
- Calm
- High-contrast
- Apple / Google-inspired simplicity

Rules:
- One to two fonts only
- Tech inspired accents | soothing and confident
- No gradients
- No animations (MVP)
- No visual clutter

Interaction:
- Minimal clicks
- Clear hierarchy
- Content first, decoration later

If unsure, remove elements rather than add.
